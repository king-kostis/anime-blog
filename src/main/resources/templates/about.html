About the Author

Hello! My name is Jesse Aaron Quartey, a third-year Data Science and Analytics student with a growing passion for backend development and software engineering. This website is my first full-stack project — a simple but meaningful step in applying everything I’ve been learning on my own.

Anime News is an RSS-powered anime blog that automatically streams updates from a Telegram channel called Anime News. The goal was to create something that connects my interest in anime with my excitement for building functional web applications.

The project was built using:

  • Java Spring Boot – for backend logic and RSS feed processing
  • Thymeleaf – as the template engine for dynamic and server-side rendering
  • Tailwind CSS – for modern, responsive, and minimal styling
  • HTML, CSS, and JavaScript – for structure, presentation, and interactivity

This application also includes an integrated email service that allows users to contact me directly through the app, as well as an RSS feed parser that fetches and displays the latest anime-related news in real time.

This was a week-long project and I hope to improve on it in the future.

Thank you for visiting the site.
